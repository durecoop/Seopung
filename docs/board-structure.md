# 서풍 홈페이지 게시판 구조 설계

## 1. 게시판 전체 구조

```
게시판 (Board)
├── 공지사항 (Notice)
├── 뉴스/보도자료 (News)
├── 자료실 (Resources)
├── FAQ (자주 묻는 질문)
├── 제품 문의 (Product Inquiry) - 비공개
├── 견적 요청 (Quote Request) - 비공개
├── 채용정보 (Careers)
└── 파트너 후기 (Partner Reviews)
```

---

## 2. 게시판별 상세 설계

### 2.1 공지사항 (Notice)
| 항목 | 내용 |
|------|------|
| **URL** | `/board/notice.html` |
| **용도** | 회사 공지, 휴무일, 정책 변경 |
| **공개여부** | 공개 |
| **작성권한** | 관리자만 |
| **댓글** | 없음 |

**필드 구성:**
- 제목 (필수)
- 내용 (필수, 에디터)
- 첨부파일 (선택, 최대 3개)
- 중요공지 여부 (상단 고정)
- 작성일
- 조회수

---

### 2.2 뉴스/보도자료 (News)
| 항목 | 내용 |
|------|------|
| **URL** | `/board/news.html` |
| **용도** | 언론보도, 수상, 인증 획득 |
| **공개여부** | 공개 |
| **작성권한** | 관리자만 |
| **댓글** | 없음 |

**필드 구성:**
- 제목 (필수)
- 카테고리: 언론보도 / 수상 / 인증 / 기타
- 썸네일 이미지 (선택)
- 내용 (필수, 에디터)
- 출처/언론사명 (선택)
- 원문 링크 (선택)
- 작성일
- 조회수

---

### 2.3 자료실 (Resources)
| 항목 | 내용 |
|------|------|
| **URL** | `/board/resources.html` |
| **용도** | 카탈로그, 인증서, 품질성적서 |
| **공개여부** | 공개 (일부 회원전용 가능) |
| **작성권한** | 관리자만 |
| **댓글** | 없음 |

**필드 구성:**
- 제목 (필수)
- 카테고리: 제품카탈로그 / 인증서 / 품질성적서 / 회사소개서
- 설명 (선택)
- 첨부파일 (필수, PDF/이미지)
- 파일 용량 표시
- 다운로드 수
- 작성일

**카테고리별 예시 자료:**
- 제품카탈로그: 냉동수산물 카탈로그 2024
- 인증서: HACCP, ASC, MSC 인증서
- 품질성적서: 월별 품질검사 성적서
- 회사소개서: 서풍 회사소개서 (국문/영문)

---

### 2.4 FAQ (자주 묻는 질문)
| 항목 | 내용 |
|------|------|
| **URL** | `/board/faq.html` |
| **용도** | 자주 묻는 질문과 답변 |
| **공개여부** | 공개 |
| **작성권한** | 관리자만 |
| **형식** | 아코디언 (접기/펼치기) |

**필드 구성:**
- 질문 (필수)
- 답변 (필수)
- 카테고리: 거래문의 / 제품정보 / 배송/물류 / 인증/품질 / 기타
- 정렬순서

**예시 FAQ:**
```
[거래문의]
Q: 최소 주문 수량은 어떻게 되나요?
A: 제품별로 상이하며, 일반적으로 박스 단위로 주문 가능합니다.
   자세한 내용은 제품 문의를 통해 상담해 주세요.

[배송/물류]
Q: 전국 배송이 가능한가요?
A: 네, 전국 냉동/냉장 배송이 가능합니다.
   도서산간 지역은 별도 문의 부탁드립니다.

[인증/품질]
Q: HACCP 인증을 받은 제품인가요?
A: 네, 당사는 HACCP 인증 시설에서 전 제품을 생산하고 있습니다.
```

### 2.5 제품 문의 (Product Inquiry)
| 항목 | 내용 |
|------|------|
| **URL** | `/board/inquiry.html` |
| **용도** | B2B 고객 제품 문의 |
| **공개여부** | 비공개 (본인/관리자만) |
| **작성권한** | 누구나 |
| **댓글** | 관리자 답변 |

**필드 구성:**
- 문의 유형: 제품문의 / 샘플요청 / 거래조건 / 기타
- 회사명 (필수)
- 담당자명 (필수)
- 연락처 (필수)
- 이메일 (필수)
- 제목 (필수)
- 문의 내용 (필수)
- 첨부파일 (선택)
- 개인정보 수집 동의 (필수)
- 답변 상태: 대기중 / 답변완료

---

### 2.6 견적 요청 (Quote Request)
| 항목 | 내용 |
|------|------|
| **URL** | `/board/quote.html` |
| **용도** | 대량 주문 견적 요청 |
| **공개여부** | 비공개 (본인/관리자만) |
| **작성권한** | 누구나 |
| **댓글** | 관리자 답변 |

**필드 구성:**
- 회사명 (필수)
- 사업자등록번호 (선택)
- 담당자명 (필수)
- 연락처 (필수)
- 이메일 (필수)
- 주소 (선택)
- 희망 제품 (체크박스 다중선택)
  - [ ] 노르웨이 간고등어
  - [ ] 손질 오징어
  - [ ] 냉동 참돔
  - [ ] 기타 (직접입력)
- 예상 물량 (필수)
- 희망 납품일 (선택)
- 추가 요청사항 (선택)
- 개인정보 수집 동의 (필수)
- 처리 상태: 접수 / 검토중 / 견적발송 / 완료

## 3. 게시판 공통 기능

### 3.1 목록 페이지
- 페이지네이션 (10/20/30개씩 보기)
- 검색 (제목, 내용, 제목+내용)
- 카테고리 필터 (해당 게시판)
- 정렬 (최신순, 조회순)

### 3.2 상세 페이지
- 이전글/다음글 네비게이션
- 목록으로 돌아가기
- 인쇄하기 (선택)
- SNS 공유 (선택)

### 3.3 관리자 기능
- 글 작성/수정/삭제
- 공지 상단고정
- 답변 작성 (문의 게시판)
- 상태 변경 (견적요청, 채용)
- 일괄 삭제

## 7. 기술 고려사항

### 정적 사이트 (GitHub Pages)의 경우
- 게시판 기능 구현을 위해 외부 서비스 연동 필요
- 추천 옵션:
  1. **Google Forms + Sheets**: 문의/견적 접수
  2. **Disqus**: 댓글 시스템
  3. **Netlify Forms**: 폼 제출
  4. **Airtable**: 데이터베이스

### 동적 사이트로 전환 시
- 백엔드: Node.js, PHP, Python 등
- 데이터베이스: MySQL, PostgreSQL, MongoDB
- CMS: WordPress, Strapi, Ghost 등

## 8. 디자인 가이드라인

### 게시판 리스트
- 테이블 형식 (PC) / 카드 형식 (모바일)
- 호버 효과
- 상단고정 공지 강조 (배경색 또는 아이콘)

### 게시판 상세
- 깔끔한 타이포그래피
- 충분한 여백
- 첨부파일 다운로드 버튼 명확히

### 폼 디자인
- 필수 항목 * 표시
- 실시간 유효성 검사
- 제출 후 확인 메시지

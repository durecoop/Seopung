3. 관리자 페이지 사용법

3.1 관리자 페이지 접속

  1. 브라우저에서 https://durecoop.github.io/Seopung/admin/ 접속
  2. 로그인 페이지에서 관리자 계정으로 로그인
  3. 대시보드에서 각 메뉴 선택

3.2 관리자 메뉴 구성

메뉴 파일명 기능
대시보드 admin/dashboard.html 전체 현황 요약
공지사항 관리 admin/notices.html 공지사항 목록/작성/수정/삭제
뉴스 관리 admin/news-manage.html 뉴스 목록/작성/수정/삭제
자료실 관리 admin/resources-manage.html 자료 업로드/수정/삭제
제품 관리 admin/products-manage.html 제품 등록/수정/삭제
갤러리 관리 admin/gallery-manage.html 갤러리 이미지 관리
FAQ 관리 admin/faq-manage.html FAQ 등록/수정/삭제
문의 확인 admin/inquiries.html 고객 문의 확인/답변
견적 확인 admin/quotes.html 견적 요청 확인/처리
샘플 데이터 admin/seed-data.html 테스트 데이터 등록

3.3 콘텐츠 등록 방법

공지사항 등록

  1. 관리자 > 공지사항 관리 클릭
  2. "새 공지 작성" 버튼 클릭
  3. 제목, 내용 입력
  4. 중요 공지 여부 체크
  5. "등록" 버튼 클릭

제품 등록

  1. 관리자 > 제품 관리 클릭
  2. "제품 추가" 버튼 클릭
  3. 제품명, 카테고리, 원산지, 규격 입력
  4. 제품 이미지 업로드
  5. "저장" 버튼 클릭

자료실 파일 업로드

  1. 관리자 > 자료실 관리 클릭
  2. "자료 등록" 버튼 클릭
  3. 자료명, 카테고리, 설명 입력
  4. 파일 드래그 앤 드롭 또는 선택
  5. "업로드" 버튼 클릭

4. 콘텐츠 관리 방법

4.1 이미지 권장 사양

용도 권장 크기 파일 형식 최대 용량
제품 이미지 800 x 800 px JPG, PNG 500KB
갤러리 이미지 1200 x 800 px JPG, PNG 1MB
뉴스 썸네일 800 x 600 px JPG, PNG 300KB
배너 이미지 1920 x 1080 px JPG 2MB

4.2 자료실 파일 카테고리

  • 카탈로그: 제품 카탈로그, 가격표
  • 인증서: HACCP, ASC, MSC, 사업자등록증 등
  • 품질성적서: 방사능 검사, 중금속 검사, 미생물 검사
  • 회사소개서: 회사 소개 자료

4.3 뉴스 카테고리

  • 언론보도: 외부 언론 기사
  • 수상: 수상 소식
  • 인증: 인증 획득 소식
  • 기타: 회사 소식

5. Firebase 데이터베이스

5.1 Firebase 콘솔 접속

  1. https://console.firebase.google.com/ 접속
  2. Google 계정으로 로그인
  3. "seopung-website" 프로젝트 선택

5.2 데이터베이스 컬렉션

컬렉션명 용도 주요 필드
notices 공지사항 title, content, important, createdAt
news 뉴스/보도자료 title, content, category, thumbnail, source
resources 자료실 title, category, url, format, size
products 제품 name, category, origin, image, spec
gallery 갤러리 title, category, image, description
faq FAQ question, answer, category, order
inquiries 문의 name, phone, email, message, status
quotes 견적요청 company, product, quantity, status
careers 채용 title, department, type, description
참고: Firebase 무료 티어(Spark Plan)는 일일 읽기 50,000회, 쓰기 20,000회로 제한됩니다. 일반적인 기업 홈페이지 운영에는 충분합니다.

6. 유지보수 및 업데이트

6.1 소스 코드 수정 방법

  1. GitHub 저장소 접속: https://github.com/durecoop/Seopung
  2. 수정할 파일 선택
  3. 연필 아이콘(Edit) 클릭
  4. 내용 수정 후 "Commit changes" 클릭
  5. 약 1~2분 후 변경사항 반영

6.2 주요 파일 위치

파일/폴더 설명
index.html 메인 페이지
css/style.css 기본 스타일
css/responsive.css 반응형 스타일
js/main.js 메인 JavaScript
admin/ 관리자 페이지 폴더
board/ 게시판 페이지 폴더

6.3 정기 점검 사항

  • 주간: 문의/견적 요청 확인 및 처리
  • 월간: 뉴스, 공지사항 업데이트
  • 분기: 제품 정보 업데이트, 인증서 갱신 확인
  • 연간: 회사 정보(매출, 연혁 등) 업데이트

7. 권장 촬영 콘텐츠

7.1 사진 촬영 권장 목록

시설 사진

촬영 대상 촬영 포인트 활용처
공장 외관 정면, 전경, 간판 포함 회사소개, 메인
가공 라인 IQF 설비, 포장 라인, 선별 공정 가공프로세스
냉동 창고 대형 냉동고 내부, 적재 현황 갤러리
품질 검사실 검사 장비, 검사 과정 가공프로세스
위생 시설 에어샤워, 손세척 시설, HACCP 표지 회사소개
사무실 사무 공간, 회의실 채용정보

제품 사진

촬영 대상 촬영 포인트 활용처
개별 제품 포장 상태, 흰 배경, 정면 제품 카탈로그
제품 단면 살 색깔, 신선도 강조 제품 상세
박스 포장 출하 박스, 라벨 포함 B2B 제품
조리 예시 구이, 조림, 회 등 완성 요리 제품 홍보

작업 현장 사진

촬영 대상 촬영 포인트 활용처
원료 입고 신선한 원료, 트럭 하역 가공프로세스
선별 작업 숙련된 직원의 선별 장면 가공프로세스
가공 작업 손질, 절단, 포장 과정 갤러리
출하 작업 완제품 적재, 배송 차량 갤러리
팀 사진 직원 단체사진, 작업복 착용 회사소개

7.2 영상 촬영 권장 목록

영상 주제 권장 길이 촬영 내용 활용처
회사 소개 영상 2~3분 회사 전경, 시설, 가공 과정, 직원 인터뷰 메인 배경, 회사소개
가공 프로세스 1~2분 원료 입고 → 선별 → 가공 → 포장 → 출하 가공프로세스
제품별 소개 30초~1분 제품 특징, 조리 예시 제품 상세
시설 투어 2~3분 공장 내부 워킹 투어 갤러리, B2B 영업
드론 촬영 30초~1분 공장 전경, 여수 바다, 항구 메인 배경
촬영 팁:
  • 가로 촬영 권장 (16:9 비율)
  • 밝은 조명에서 촬영
  • 작업복, 위생모 등 착용하여 청결함 강조
  • HACCP, 인증 마크가 보이도록 촬영
  • 4K 해상도 권장 (최소 Full HD)

8. 문제 해결 가이드

8.1 자주 발생하는 문제

문제 원인 해결 방법
페이지가 업데이트되지 않음 브라우저 캐시 Ctrl+Shift+R (강력 새로고침)
이미지가 안 보임 파일 경로 오류 파일명 확인, 대소문자 구분
데이터가 로딩되지 않음 Firebase 연결 문제 Firebase 콘솔에서 상태 확인
관리자 로그인 안됨 계정 권한 문제 Firebase Authentication 확인
파일 업로드 실패 용량 초과 또는 형식 오류 파일 크기/형식 확인
